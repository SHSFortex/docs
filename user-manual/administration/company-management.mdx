# Company Management

Company Admins can configure organization settings, branding, and preferences.

## Accessing Company Settings

<Steps>
  <Step title="Navigate to Administration">
    Click **Administration** in the sidebar menu
  </Step>

  <Step title="Select Company Settings">
    Choose **Company Management** or **Settings**
  </Step>

  <Step title="View/Edit Settings">
    Update configuration as needed
  </Step>

  <Step title="Save Changes">
    Click **Save** to apply changes
  </Step>
</Steps>

## Branding & Customization

### Logo Upload

Upload your organization's logo:

1. Go to **Branding** section
2. Click **Upload Logo**
3. Select image file (PNG, JPG, SVG)
4. Recommended size: 200x200px minimum
5. Preview and save

<Note>
Logos appear in the navigation header and participant views.
</Note>

### Color Scheme

Customize the platform's appearance:

- **Primary Color**: Main brand color
- **Secondary Color**: Accent color
- **Background**: Custom background (optional)

### Welcome Message

Set a custom welcome message for users:

- Shown on login page
- Displayed on dashboard
- Supports markdown formatting
- Multilingual (DE/EN)

## User Settings

### Default User Role

Choose the default role for new users:

- **Participant** (most common)
- **Content Creator** (for training organizations)
- Custom role (if configured)

<Warning>
Default roles apply to users joining via join codes. Email invitations can specify individual roles.
</Warning>

### Email Verification

Configure email verification requirements:

- **Required for All**: All users must verify email
- **Required for Participants**: Only participants need verification
- **Optional**: Verification not enforced (not recommended)

### User Invitation Settings

- **Who Can Invite**: Admins only or all users
- **Invitation Expiry**: Days until invitation expires (default: 7)
- **Join Code Duration**: Validity period for join codes

  <Tab title="Scoring">

    - Pass/fail threshold

  </Tab>

### Scenario Library

Manage organization scenario access:

- **Public Scenarios**: Available to all organizations
- **Private Scenarios**: Only your organization
- **Shared Scenarios**: Shared with partner organizations

## Team Management

### Creating Teams

Organize users into teams for group simulations:

<Steps>
  <Step title="Go to Teams">
    Navigate to **Administration** > **Teams**
  </Step>

  <Step title="Create New Team">
    Click **+ Create Team**
  </Step>

  <Step title="Name and Describe">
    Provide team name and description
  </Step>

  <Step title="Add Members">
    Select users to add to the team
  </Step>

  <Step title="Assign Team Lead">
    Optionally designate a team leader
  </Step>

  <Step title="Save Team">
    Click **Create** to save
  </Step>
</Steps>

### Team Features

- **Team Simulations**: Assign simulations to entire teams
- **Team Reports**: Aggregate performance analytics
- **Team Communication**: Dedicated team channels
- **Team Resources**: Shared documents and materials

## Company Archive

### Archiving the Company

If you need to deactivate your organization:

<Warning>
Archiving a company disables all user access and simulations. This action requires confirmation and may not be reversible without contacting support.
</Warning>

<Steps>
  <Step title="Navigate to Settings">
    Go to **Company Settings** > **Advanced**
  </Step>

  <Step title="Request Archive">
    Click **Archive Company**
  </Step>

  <Step title="Confirm Action">
    Type company name to confirm
  </Step>

  <Step title="Archive Applied">
    Company is deactivated; users cannot log in
  </Step>
</Steps>
