<Check>Real-time simulation events</Check>
<Check>Team messages</Check>
<Check>Decision deadlines</Check>
<Check>System alerts</Check>

### Push Notifications

For PWA installation:

- Simulation starting soon
- Your turn in simulation
- Results available

**Configure:** Allow in browser when prompted, or in system settings

## Data & Performance

### Bandwidth Optimization

For slower connections:

- **Reduce Image Quality**: Faster loading
- **Disable Auto-Play**: Videos don't auto-start
- **Limit Animations**: Simpler UI transitions
- **Compact Mode**: Less data per page

### Storage

Clear cached data:

- **Clear Cache**: Free up space
- **Offline Data**: Manage PWA stored data
- **Media Downloads**: Remove downloaded resources

**Manage:** Profile > Integrations

## Resetting Preferences

### Reset to Defaults

Return all settings to original:

1. Profile > Preferences
2. Scroll to bottom
3. Click **Reset to Defaults**
4. Confirm action
5. Preferences restored

<Note>
Does not affect profile info, only preferences. Your training data is preserved.
</Note>
