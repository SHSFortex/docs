# User Profile

Your user profile contains personal information, preferences, and settings for your crisisIQ account.

## Accessing Your Profile

<Steps>
  <Step title="Click Profile Icon">
    Top-right corner of dashboard
  </Step>

  <Step title="Select 'Profile' or 'Account Settings'">
    From dropdown menu
  </Step>

  <Step title="View/Edit Information">
    Update as needed
  </Step>

  <Step title="Save Changes">
    Click **Save** or **Update Profile**
  </Step>
</Steps>

### Profile Photo

Upload a profile picture:

1. Click **Change Photo** or camera icon
2. Select image file (JPG, PNG)
3. Crop/adjust if needed
4. Save

**Recommendations:**
- Professional headshot
- Clear face visibility
- Neutral background
- 200x200px minimum
- Under 5MB file size

### Bio/About

Optional personal or professional bio:

- Brief introduction
- Areas of expertise
- Crisis management interests
- Contact preferences

## Training History

### Your Record

Profile displays:

- **Scenarios Completed**: Total count
- **Average Score**: Overall performance
- **Certificates Earned**: Achievements
- **Training Hours**: Time invested
- **Recent Activity**: Latest simulations

### Viewing Full History

Access complete training record:

1. **Profile** > **Training History**
2. See all completed scenarios
3. Filter by date, type, score
4. Export transcript if needed

## Security Settings

### Password Management

<Steps>
  <Step title="Navigate to Security">
    Profile > Security or Password section
  </Step>

  <Step title="Enter Current Password">
    For verification
  </Step>

  <Step title="Enter New Password">
    Meet security requirements
  </Step>

  <Step title="Confirm New Password">
    Re-type to verify
  </Step>

  <Step title="Save">
    Password updated, re-login may be required
  </Step>
</Steps>

<Card title="Password Requirements" icon="key" href="/user-manual/authentication/password-reset">
  See password guidelines
</Card>

### Two-Factor Authentication

Enable 2FA for enhanced security:

<Steps>
  <Step title="Go to Security Settings">
    Profile > Security > Two-Factor Authentication
  </Step>

  <Step title="Click 'Enable 2FA'">
    Start setup process
  </Step>

  <Step title="Scan QR Code">
    Use authenticator app (Google Authenticator, Authy, etc.)
  </Step>

  <Step title="Enter Verification Code">
    From authenticator app
  </Step>

  <Step title="Save Backup Codes">
    Store securely in case you lose device
  </Step>

  <Step title="2FA Activated">
    Required on next login
  </Step>
</Steps>

### Active Sessions

Monitor logged-in devices:

- See all active sessions

- **Log Out** from specific devices remotely

## Privacy Controls

### Data Visibility

Control who sees your information:

<Tabs>
  <Tab title="Profile Visibility">
    **Who Can See Your Profile**

    - Only Me
    - My Organization
    - All crisisIQ Users
  </Tab>

  <Tab title="Performance Sharing">
    **Who Can See Your Scores**

    - Private (only you and admins)
    - Team members (in shared simulations)
    - Organization
    - Public leaderboards (opt-in)
  </Tab>

  <Tab title="Activity">
    **Show Online Status**

    - Display when you're active
    - Hide activity from others
    - Show last login time
  </Tab>
</Tabs>

### Data Management

<AccordionGroup>
  <Accordion title="Export Your Data">
    Download all your personal data:
    - Profile information
    - Training history
    - Scores and feedback
    - Certificates

    **Profile** > **Privacy** > **Export Data**
  </Accordion>

  <Accordion title="Delete Account">
    Permanently remove account:

    <Warning>
    This action cannot be undone. All your data will be deleted.
    </Warning>

    **Profile** > **Privacy** > **Delete Account**
    - Requires confirmation
    - May need to contact admin
    - Some data retained for compliance
  </Accordion>
</AccordionGroup>
