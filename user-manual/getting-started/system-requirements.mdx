# Systemanforderungen

Um eine optimale Erfahrung mit crisisIQ zu gewährleisten, beachten Sie bitte die folgenden Systemanforderungen.

crisisIQ ist eine moderne Webanwendung, die am besten mit aktuellen Browsern funktioniert.

- **iOS Safari**: iOS 14 oder höher
- **Chrome Mobile**: Neueste Version
- **Firefox Mobile**: Neueste Version
- **Samsung Internet**: Neueste Version

<Note>
Für die beste mobile Erfahrung empfehlen wir die Installation von crisisIQ als Progressive Web App (PWA). Details finden Sie in unserer [Mobile Installationsanleitung](/user-manual/mobile/ios-installation).
</Note>

## Geräteanforderungen

### Desktop/Laptop
- **Betriebssystem**: Windows 10+, macOS 10.15+ oder Linux (jede moderne Distribution)
- **Prozessor**: 2 GHz Dual-Core oder besser
- **RAM**: 4 GB Minimum (8 GB empfohlen)
- **Bildschirmauflösung**: 1280x720 Minimum (1920x1080 empfohlen)

### Tablet
- **iOS**: iPad mit iOS 14 oder höher
- **Android**: Android 10 oder höher
- **Bildschirmgröße**: 9,7 Zoll oder größer empfohlen

### Smartphone
- **iOS**: iPhone mit iOS 14 oder höher
- **Android**: Android 10 oder höher
- **Bildschirmgröße**: 5 Zoll oder größer

## Internetverbindung

<Warning>
Eine stabile Internetverbindung ist erforderlich, damit crisisIQ ordnungsgemäß funktioniert.
</Warning>

### Mindestanforderungen
- **Download-Geschwindigkeit**: 5 Mbps
- **Upload-Geschwindigkeit**: 1 Mbps
- **Latenz**: Weniger als 100ms

### Empfohlen
- **Download-Geschwindigkeit**: 25 Mbps oder höher
- **Upload-Geschwindigkeit**: 5 Mbps oder höher
- **Latenz**: Weniger als 50ms

<Info>
Echtzeit-Funktionen (Live-Simulationen, Chat, KI-Mentor) erfordern eine konstante Verbindung. WLAN oder kabelgebundene Verbindungen werden gegenüber mobilen Datenverbindungen empfohlen.
</Info>

## Zusätzliche Anforderungen

### JavaScript
JavaScript muss in Ihrem Browser aktiviert sein. crisisIQ ist eine moderne Single-Page-Anwendung, die JavaScript für den Betrieb benötigt.

### Cookies & Local Storage
Ihr Browser muss Cookies und Local Storage für crisisIQ zulassen, um Ihren Login-Status und Ihre Einstellungen zu speichern.

### Pop-ups
Bitte erlauben Sie Pop-ups von crisisIQ, um sicherzustellen, dass Funktionen wie Dokumentanzeige und Zertifikat-Downloads korrekt funktionieren.

## Optionale Funktionen

### Text-to-Speech (TTS)
- Unterstützt in allen modernen Browsern
- Erfordert Browser-Berechtigung für Audio-Wiedergabe
- Wird für Ereignisbenachrichtigungen und Barrierefreiheitsfunktionen verwendet

### Benachrichtigungen
- Push-Benachrichtigungen verfügbar bei Installation als PWA
- Browser-Benachrichtigungen für wichtige Updates
- Erfordert Benutzererlaubnis

### Offline-Modus
- Verfügbar bei Installation als PWA
- Eingeschränkte Funktionalität ohne Internetverbindung
- Zwischengespeicherte Szenarien und Ressourcen offline zugänglich

## Barrierefreiheit

crisisIQ ist mit Blick auf Barrierefreiheit konzipiert:

- **Screen Reader**: Kompatibel mit NVDA, JAWS, VoiceOver
- **Tastaturnavigation**: Vollständige Tastaturunterstützung
- **Hoher Kontrast**: Unterstützt systemweite Hochkontrast-Modi
- **Schriftskalierung**: Berücksichtigt Browser-Zoom und Schriftgrößeneinstellungen

## Netzwerk & Firewall

Bei Verwendung von crisisIQ in einer Unternehmensumgebung stellen Sie Folgendes sicher:

### Erforderliche Domains
Erlauben Sie den Zugriff auf diese Domains in Ihrer Firewall:
- `crisisiq.com` (Hauptanwendung)
- `api.crisisiq.com` (Backend-API)
- `ws.crisisiq.com` (WebSocket für Echtzeit-Funktionen)

### Ports
- **HTTPS**: Port 443 (erforderlich)
- **WebSocket**: Port 443 über WSS-Protokoll (erforderlich für Echtzeit-Funktionen)

## Performance-Tipps

<AccordionGroup>
  <Accordion title="Unnötige Browser-Tabs schließen">
    Viele geöffnete Tabs können die Performance verlangsamen. Schließen Sie nicht verwendete Tabs während der Ausführung von Simulationen.
  </Accordion>

  <Accordion title="Browser-Erweiterungen deaktivieren">
    Einige Browser-Erweiterungen können mit crisisIQ interferieren. Versuchen Sie, diese zu deaktivieren, wenn Sie Probleme haben.
  </Accordion>

  <Accordion title="Browser-Cache leeren">
    Wenn Sie veraltete Inhalte oder Fehler bemerken, leeren Sie Ihren Browser-Cache und laden Sie die Seite neu.
  </Accordion>

  <Accordion title="Kabelgebundene Verbindung verwenden">
    Für die stabilste Erfahrung während Live-Simulationen verwenden Sie eine kabelgebundene Ethernet-Verbindung anstelle von WLAN.
  </Accordion>
</AccordionGroup>

## Fehlerbehebung

Bei technischen Problemen:

1. **Aktualisieren Sie Ihren Browser** auf die neueste Version
2. **Leeren Sie Cache und Cookies** für crisisiq.com
3. **Deaktivieren Sie Browser-Erweiterungen** vorübergehend
4. **Versuchen Sie einen anderen Browser** um das Problem zu isolieren
5. **Überprüfen Sie Ihre Internetverbindung** auf Geschwindigkeit und Stabilität
6. **Kontaktieren Sie den Support** unter support@crisisiq.com mit Details zu Ihrem System
