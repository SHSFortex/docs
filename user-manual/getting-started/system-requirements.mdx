# System Requirements

To ensure the best experience with crisisIQ, please review the following system requirements.

## Supported Browsers

crisisIQ is a modern web application that works best with up-to-date browsers.

### Desktop Browsers

  </Card>
  <Card title="Safari" icon="safari">

  </Card>

### Mobile Browsers

- **iOS Safari**: iOS 14 or later
- **Chrome Mobile**: Latest version
- **Firefox Mobile**: Latest version
- **Samsung Internet**: Latest version

<Note>
For the best mobile experience, we recommend installing crisisIQ as a Progressive Web App (PWA). See our [Mobile Installation Guide](/user-manual/mobile/ios-installation) for details.
</Note>

## Device Requirements

### Desktop/Laptop
- **Operating System**: Windows 10+, macOS 10.15+, or Linux (any modern distribution)
- **Processor**: 2 GHz dual-core or better
- **RAM**: 4 GB minimum (8 GB recommended)
- **Screen Resolution**: 1280x720 minimum (1920x1080 recommended)

### Tablet
- **iOS**: iPad running iOS 14 or later
- **Android**: Android 10 or later
- **Screen Size**: 9.7 inches or larger recommended

### Smartphone
- **iOS**: iPhone running iOS 14 or later
- **Android**: Android 10 or later
- **Screen Size**: 5 inches or larger

## Internet Connection

<Warning>
A stable internet connection is required for crisisIQ to function properly.
</Warning>

### Minimum Requirements
- **Download Speed**: 5 Mbps
- **Upload Speed**: 1 Mbps
- **Latency**: Less than 100ms

### Recommended
- **Download Speed**: 25 Mbps or higher
- **Upload Speed**: 5 Mbps or higher
- **Latency**: Less than 50ms

<Info>
Real-time features (live simulations, chat, AI mentor) require a consistent connection. WiFi or wired connections are recommended over cellular data.
</Info>

## Additional Requirements

### JavaScript
JavaScript must be enabled in your browser. crisisIQ is a modern single-page application that requires JavaScript to function.

### Cookies & Local Storage
Your browser must allow cookies and local storage for crisisIQ to save your login state and preferences.

### Pop-ups
Please allow pop-ups from crisisIQ to ensure features like document viewing and certificate downloads work correctly.

## Optional Features

### Text-to-Speech (TTS)
- Supported in all modern browsers
- Requires browser permission for audio playback
- Used for event notifications and accessibility features

### Notifications
- Push notifications available when installed as PWA
- Browser notifications for important updates
- Requires user permission

### Offline Mode
- Available when installed as PWA
- Limited functionality without internet connection
- Cached scenarios and resources accessible offline

## Accessibility

crisisIQ is designed with accessibility in mind:

- **Screen Readers**: Compatible with NVDA, JAWS, VoiceOver
- **Keyboard Navigation**: Full keyboard support
- **High Contrast**: Supports system-level high contrast modes
- **Font Scaling**: Respects browser zoom and font size settings

## Network & Firewall

If you're using crisisIQ in a corporate environment, ensure the following:

### Required Domains
Allow access to these domains in your firewall:
- `crisisiq.com` (main application)
- `api.crisisiq.com` (backend API)
- `ws.crisisiq.com` (WebSocket for real-time features)

### Ports
- **HTTPS**: Port 443 (required)
- **WebSocket**: Port 443 via WSS protocol (required for real-time features)

## Performance Tips

<AccordionGroup>
  <Accordion title="Close unnecessary browser tabs">
    Having many tabs open can slow down performance. Close tabs you're not using while running simulations.
  </Accordion>

  <Accordion title="Disable browser extensions">
    Some browser extensions may interfere with crisisIQ. Try disabling them if you experience issues.
  </Accordion>

  <Accordion title="Clear browser cache">
    If you notice outdated content or errors, clear your browser cache and reload the page.
  </Accordion>

  <Accordion title="Use a wired connection">
    For the most stable experience during live simulations, use a wired ethernet connection instead of WiFi.
  </Accordion>
</AccordionGroup>

## Troubleshooting

If you're experiencing technical issues:

1. **Update your browser** to the latest version
2. **Clear cache and cookies** for crisisiq.com
3. **Disable browser extensions** temporarily
4. **Try a different browser** to isolate the issue
5. **Check your internet connection** speed and stability
6. **Contact support** at support@crisisiq.com with details about your system
