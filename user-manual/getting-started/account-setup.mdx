# Kontoeinrichtung

Beginnen Sie mit crisisIQ, indem Sie Ihr Konto erstellen und den ersten Einrichtungsprozess abschließen.

## Ihr Konto erstellen

Es gibt drei Möglichkeiten, ein crisisIQ-Konto zu erstellen:

  <Card title="E-Mail-Einladung" icon="envelope">

## Methode 1: Direkte Registrierung

<Steps>
  <Step title="Login-Seite aufrufen">
    Navigieren Sie zur crisisIQ-Login-Seite unter der URL Ihrer Organisation oder der öffentlichen Instanz.
  </Step>

  <Step title="Auf 'Konto erstellen' klicken">
    Wählen Sie die Registrierungsoption, um den Anmeldeprozess zu starten.
  </Step>

  <Step title="Ihre Informationen eingeben">
    Geben Sie die folgenden Details an:
    - **Vollständiger Name**: Ihr Vor- und Nachname
    - **E-Mail-Adresse**: Eine gültige E-Mail, auf die Sie zugreifen können
    - **Passwort**: Muss die Sicherheitsanforderungen erfüllen (siehe unten)
    - **Unternehmen** (falls zutreffend): Wählen oder geben Sie Ihre Organisation ein
  </Step>

  <Step title="E-Mail verifizieren">
    Überprüfen Sie Ihren Posteingang auf eine Verifizierungs-E-Mail und klicken Sie auf den Bestätigungslink.
  </Step>

  <Step title="Ihr Profil vervollständigen">
    Fügen Sie zusätzliche Informationen wie Profilbild, Rolle und Präferenzen hinzu.
  </Step>
</Steps>

### Passwortanforderungen

Ihr Passwort muss diese Kriterien erfüllen:

<Check>Mindestens 8 Zeichen lang</Check>
<Check>Mindestens ein Großbuchstabe</Check>
<Check>Mindestens ein Kleinbuchstabe</Check>
<Check>Mindestens eine Zahl</Check>
<Check>Mindestens ein Sonderzeichen (!@#$%^&*)</Check>

<Warning>
Wählen Sie ein starkes, einzigartiges Passwort. Vermeiden Sie persönliche Informationen oder gebräuchliche Wörter.
</Warning>

## Methode 2: E-Mail-Einladung

Wenn Ihr Organisationsadministrator Sie eingeladen hat:

<Steps>
  <Step title="Ihre E-Mail überprüfen">
    Suchen Sie nach einer Einladungs-E-Mail von crisisIQ mit dem Betreff "Sie wurden eingeladen, [Firmenname] auf crisisIQ beizutreten"
  </Step>

  <Step title="Auf den Einladungslink klicken">
    Die E-Mail enthält einen sicheren Link zum Annehmen der Einladung. Klicken Sie darauf, um fortzufahren.
  </Step>

  <Step title="Ihr Passwort festlegen">
    Erstellen Sie ein Passwort für Ihr Konto (muss die Sicherheitsanforderungen erfüllen)
  </Step>

  <Step title="Ihre E-Mail verifizieren">
    Falls erforderlich, verifizieren Sie Ihre E-Mail-Adresse, indem Sie auf den Bestätigungslink klicken, der Ihnen zugesandt wurde
  </Step>

  <Step title="Auf Ihr Konto zugreifen">
    Nach der Verifizierung werden Sie eingeloggt und können auf Ihre zugewiesene Rolle und Simulationen zugreifen
  </Step>
</Steps>

<Info>
Einladungslinks laufen nach 7 Tagen ab. Wenn Ihr Link abgelaufen ist, kontaktieren Sie Ihren Administrator, um die Einladung erneut zu senden.
</Info>

## Methode 3: Mit Code beitreten

Einige Organisationen verwenden Beitrittscodes für schnellen Zugriff:

<Steps>
  <Step title="Zur Beitrittsseite navigieren">
    Gehen Sie zur Option "Mit Code beitreten" auf dem Login-Bildschirm
  </Step>

  <Step title="Den Beitrittscode eingeben">
    Geben Sie den alphanumerischen Code ein, der von Ihrem Organisationsadministrator bereitgestellt wurde
  </Step>

  <Step title="Ihre Details angeben">
    Geben Sie Ihren Namen, Ihre E-Mail ein und erstellen Sie ein Passwort
  </Step>

  <Step title="Registrierung abschließen">
    Verifizieren Sie Ihre E-Mail und vervollständigen Sie die Einrichtung Ihres Profils
  </Step>
</Steps>

## Erste Anmeldung

Nach dem Erstellen Ihres Kontos erwartet Sie Folgendes bei Ihrer ersten Anmeldung:

### 1. Dashboard-Tour (Optional)

Neue Benutzer sehen möglicherweise eine Kurzeinführung, die wichtige Funktionen hervorhebt:
- Navigationsmenü
- Aktive Simulationen
- Schnellaktionen
- Profileinstellungen

### 2. Profilvervollständigung

Vervollständigen Sie Ihr Benutzerprofil mit:
- **Profilbild** (optional, aber empfohlen)
- **Berufsbezeichnung/Rolle**
- **Abteilung** (falls zutreffend)
- **Bevorzugte Sprache** (Deutsch oder Englisch)
- **Benachrichtigungspräferenzen**

### 3. Unternehmensauswahl

Wenn Sie mehreren Organisationen angehören:
1. Verwenden Sie den **Unternehmensumschalter** in der oberen Navigation
2. Wählen Sie die Organisation, mit der Sie arbeiten möchten
3. Ihr Dashboard wird aktualisiert, um die Inhalte dieses Unternehmens anzuzeigen

<Note>
Ihre Rolle und Berechtigungen können zwischen Unternehmen unterschiedlich sein. Die verfügbaren Funktionen hängen von Ihrer Rolle im ausgewählten Unternehmen ab.
</Note>

## Ihre Rolle verstehen

Ihrem Konto wird eine oder mehrere Rollen zugewiesen:

<Tabs>
  <Tab title="Teilnehmer">
    **Teilnehmer** - Simulationen beitreten und abschließen

    - Zugriff auf zugewiesene Simulationen
    - Teilnahme an Gruppenübungen
    - Solo-Training absolvieren
    - Persönliche Berichte und Zertifikate einsehen
  </Tab>

  <Tab title="Content Creator">
    **Content Creator** - Trainingsszenarien erstellen

    - Szenarien erstellen und bearbeiten
    - Timeline-Ereignisse erstellen
    - Entscheidungspunkte gestalten
    - Simulationen in der Vorschau anzeigen
  </Tab>

  <Tab title="Unternehmensadministrator">
    **Unternehmensadministrator** - Organisation verwalten

    - Benutzer und Rollen verwalten
    - Simulationen erstellen und zuweisen
    - Alle Berichte und Analysen einsehen
    - Unternehmenseinstellungen konfigurieren
  </Tab>

  <Tab title="Systemadministrator">
    **Systemadministrator** - Vollständiger Plattformzugriff

    - Alle Funktionen des Unternehmensadministrators
    - Mehrere Organisationen verwalten
    - Systemweite Konfiguration
    - Plattformverwaltung
  </Tab>
</Tabs>

## Erstkonfiguration

### Sprachpräferenz

Legen Sie Ihre bevorzugte Sprache fest:

1. Klicken Sie auf Ihr **Profilsymbol** in der oberen rechten Ecke
2. Wählen Sie **Sprache** aus dem Dropdown-Menü
3. Wählen Sie zwischen **Deutsch (DE)** oder **Englisch (EN)**
4. Die Oberfläche wird sofort aktualisiert

### Benachrichtigungseinstellungen

Konfigurieren Sie, wie Sie Updates erhalten:

- **E-Mail-Benachrichtigungen**: Simulationseinladungen, Fristen, Ergebnisse
- **In-App-Benachrichtigungen**: Echtzeit-Updates während der Anmeldung
- **Push-Benachrichtigungen**: Verfügbar bei Installation als PWA

## Kontosicherheit

### Zwei-Faktor-Authentifizierung (2FA)

Für erhöhte Sicherheit aktivieren Sie 2FA in Ihren Kontoeinstellungen:

1. Gehen Sie zu **Profil** &gt; **Sicherheitseinstellungen**
2. Klicken Sie auf **Zwei-Faktor-Authentifizierung aktivieren**
3. Scannen Sie den QR-Code mit Ihrer Authenticator-App
4. Geben Sie den Verifizierungscode zur Bestätigung ein

<Tip>
Wir empfehlen die Verwendung von Apps wie Google Authenticator, Authy oder 1Password für 2FA.
</Tip>

### Sitzungsverwaltung

- **Automatische Abmeldung**: Sitzungen laufen nach 24 Stunden Inaktivität ab
- **Mehrere Geräte**: Sie können auf mehreren Geräten gleichzeitig angemeldet sein
- **Sichere Verbindungen**: Alle Daten werden mit HTTPS/WSS verschlüsselt

## Fehlerbehebung bei der Kontoeinrichtung

<AccordionGroup>
  <Accordion title="Ich habe die Verifizierungs-E-Mail nicht erhalten">
    1. Überprüfen Sie Ihren Spam-/Junk-Ordner
    2. Stellen Sie sicher, dass Sie die richtige E-Mail-Adresse eingegeben haben
    3. Fordern Sie eine neue Verifizierungs-E-Mail von der Login-Seite an
    4. Kontaktieren Sie den Support, wenn das Problem weiterhin besteht
  </Accordion>

  <Accordion title="Mein Einladungslink ist abgelaufen">
    Kontaktieren Sie Ihren Organisationsadministrator, um die Einladung erneut zu senden. Einladungslinks sind 7 Tage gültig.
  </Accordion>

  <Accordion title="Ich kann mein Passwort nicht festlegen">
    Stellen Sie sicher, dass Ihr Passwort alle Anforderungen erfüllt:
    - Mindestens 8 Zeichen
    - Mix aus Groß-, Kleinbuchstaben, Zahlen und Sonderzeichen
    - Kein häufig verwendetes Passwort
  </Accordion>

  <Accordion title="Ich gehöre mehreren Unternehmen an">
    Verwenden Sie den Unternehmensumschalter in der oberen Navigation, um zwischen Organisationen zu wechseln. Ihre Rolle kann in jedem Unternehmen unterschiedlich sein.
  </Accordion>

  <Accordion title="E-Mail-Verifizierung funktioniert nicht">
    1. Stellen Sie sicher, dass der Link nicht abgelaufen ist (gültig für 24 Stunden)
    2. Versuchen Sie, die URL direkt in Ihren Browser zu kopieren und einzufügen
    3. Fordern Sie eine neue Verifizierungs-E-Mail an
    4. Kontaktieren Sie support@crisisiq.com für Unterstützung
  </Accordion>
</AccordionGroup>
