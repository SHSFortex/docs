# iOS Installation (iPhone/iPad)

Install crisisIQ on your iPhone or iPad for a native app experience with offline capabilities.

## What is PWA Installation?

Progressive Web App (PWA) installation provides:

<CardGroup cols={2}>
  <Card title="App-Like Experience" icon="mobile-screen">
    Runs like native app, no App Store needed
  </Card>
  <Card title="Home Screen Icon" icon="square-plus">
    Launch from home screen
  </Card>
  <Card title="Full Screen" icon="expand">
    No browser UI clutter
  </Card>
  <Card title="Offline Access" icon="wifi-slash">
    Use some features without internet
  </Card>
  <Card title="Push Notifications" icon="bell">
    Receive alerts even when app closed
  </Card>
  <Card title="Fast Loading" icon="bolt">
    Cached for quicker startup
  </Card>
</CardGroup>

## Requirements

<Check>**iOS 14 or later** (iOS 16+ recommended)</Check>
<Check>**Safari browser** (required for installation)</Check>
<Check>**Internet connection** (for initial setup)</Check>
<Check>**Storage space**: ~50-100MB</Check>

<Warning>
Installation must be done through Safari. Chrome, Firefox, or other browsers on iOS cannot install PWAs.
</Warning>

## Installation Steps

### For iPhone

<Steps>
  <Step title="Open Safari">
    Launch Safari browser (blue compass icon)
  </Step>

  <Step title="Navigate to crisisIQ">
    Go to your organization's crisisIQ URL
  </Step>

  <Step title="Log In">
    Sign in to your account
  </Step>

  <Step title="Tap Share Button">
    Tap the **Share** icon (square with up arrow) at bottom of screen
  </Step>

  <Step title="Scroll and Find 'Add to Home Screen'">
    Scroll down in the share menu and tap **"Add to Home Screen"**
  </Step>

  <Step title="Customize Name (Optional)">
    Edit the app name or keep default "crisisIQ"
  </Step>

  <Step title="Tap 'Add'">
    Confirm to add icon to home screen
  </Step>

  <Step title="Find Icon">
    Look for crisisIQ icon on your home screen
  </Step>

  <Step title="Launch">
    Tap the icon to open crisisIQ in full-screen mode
  </Step>
</Steps>

### For iPad

Same process as iPhone:

1. Open Safari
2. Navigate to crisisIQ and log in
3. Tap **Share** (top-right toolbar)
4. Select **"Add to Home Screen"**
5. Tap **Add**
6. Icon appears on home screen

## Detailed Walkthrough

### Step-by-Step with Screenshots

**1. Share Button Location**

The Share button (□↑) is located:
- **iPhone**: Bottom center of Safari
- **iPad**: Top-right of Safari toolbar

**2. Share Menu**

In the share menu:
- Scroll down to find actions
- "Add to Home Screen" has a ⊕ icon
- May need to scroll past sharing contacts

**3. Add Screen**

On the Add to Home Screen screen:
- Shows crisisIQ logo preview
- Name field (editable)
- URL shown (read-only)
- **Add** button (top-right)

**4. Home Screen**

After adding:
- Icon appears in next available slot
- Looks like native app icon
- No browser indicators
- Can be moved/organized like other apps

## Verifying Installation

### Successful Installation Indicators

<Check>crisisIQ icon on home screen</Check>
<Check>Tapping opens full-screen (no Safari UI)</Check>
<Check>Status bar shows time, battery (no URL bar)</Check>
<Check>Can multitask/switch like other apps</Check>

### If It Didn't Work

Try:
- Ensure using Safari (not Chrome or others)
- Update iOS to latest version
- Check available storage
- Try again from the beginning
- Contact support if persistent issues

## Using the Installed App

### Opening crisisIQ

Simply tap the home screen icon. No need to:
- Open Safari first
- Navigate to URL
- Deal with browser menus

### App Behavior

<Tabs>
  <Tab title="Login">
    **First Launch**

    - May need to log in again
    - "Remember Me" persists across sessions
    - Touch ID/Face ID supported (if configured)
  </Tab>

  <Tab title="Offline Mode">
    **No Internet**

    - View previously loaded content
    - Access cached scenarios
    - See your training history
    - New content requires connection
  </Tab>

  <Tab title="Updates">
    **Automatic Updates**

    - App updates automatically when online
    - No App Store updates needed
    - Always latest version
  </Tab>

  <Tab title="Notifications">
    **Push Alerts**

    - Allow notifications when prompted
    - Receive simulation reminders
    - Get results alerts
    - Manage in iOS Settings > crisisIQ
  </Tab>
</Tabs>

## Managing the App

### Moving the Icon

- Long-press icon until it jiggles
- Drag to desired location
- Press home button or swipe up to stop jiggling

### Creating Folder

- Drag crisisIQ icon onto another app
- Creates folder
- Name folder (e.g., "Training")

### Removing from Home Screen

<Steps>
  <Step title="Long-Press Icon">
    Hold until menu appears
  </Step>

  <Step title="Select 'Remove App'">
    Or "Delete" depending on iOS version
  </Step>

  <Step title="Choose 'Remove from Home Screen'">
    **Not** "Delete App" (that removes Safari bookmark)
  </Step>

  <Step title="Confirm">
    Icon removed, can re-add anytime
  </Step>
</Steps>

<Note>
Removing from home screen doesn't delete your account or data. You can still access via Safari.
</Note>

## Optimizing Mobile Experience

### Settings Recommendations

In crisisIQ app settings:

<Check>**Notifications**: Enable for timely alerts</Check>
<Check>**Offline Data**: Allow for offline access</Check>
<Check>**Cellular Data**: Enable if using outside WiFi</Check>
<Check>**Screen Brightness**: Auto-brightness recommended</Check>

### iOS Settings

In iOS Settings > crisisIQ (if listed):

- **Notifications**: Customize alert style
- **Cellular Data**: Allow/disallow mobile data use
- **Location**: If app uses location features
- **Background App Refresh**: Keep app updated

## Offline Capabilities

### What Works Offline

<CardGroup cols={2}>
  <Card title="View Content" icon="eye">
    - Previously loaded dashboard
    - Cached training history
    - Downloaded certificates
  </Card>
  <Card title="Resume Scenarios" icon="play">

### What Requires Internet

<Warning>
**Needs Connection:**
- Logging in (first time or after logout)
- Starting new simulations
- Submitting decisions
- Real-time features (chat, live events)
- Downloading new content
- Syncing progress
</Warning>

### Syncing

When you reconnect:

- Offline actions sync automatically
- Pending submissions upload
- New content downloads
- Progress updates

## Troubleshooting

<AccordionGroup>
  <Accordion title="'Add to Home Screen' option missing">
    **Causes:**
    - Not using Safari browser
    - Already added (check home screen)
    - iOS version too old

    **Solution:**
    - Switch to Safari
    - Update iOS
    - Check all home screen pages
  </Accordion>

  <Accordion title="App opens in Safari instead of full-screen">
    **Cause:** Tapping link from outside app

    **Solution:**
    - Use home screen icon to launch
    - Links from email/messages open in Safari
    - Copy URL and paste in PWA app if needed
  </Accordion>

  <Accordion title="Offline mode not working">
    **Solution:**
    - Ensure you opened app while online first
    - Check storage isn't full
    - Re-install PWA
    - Some features require internet
  </Accordion>

  <Accordion title="Notifications not appearing">
    **Solution:**
    - Check iOS Settings > Notifications > crisisIQ
    - Allow notifications
    - Ensure "Do Not Disturb" isn't active
    - Re-launch app and allow when prompted
  </Accordion>

  <Accordion title="App is slow or crashing">
    **Solution:**
    - Close and reopen app
    - Restart iPhone/iPad
    - Clear Safari cache: Settings > Safari > Clear History and Website Data
    - Re-install PWA
  </Accordion>
</AccordionGroup>
