# Dashboard Overview

The crisisIQ dashboard is your central hub for accessing simulations, managing content, and tracking your progress.

## Main Dashboard Layout

When you log in, you'll see the main dashboard with these key areas:

### Header Navigation

<Tabs>

## Dashboard Customization

### Layout Options

<Tabs>
  <Tab title="Sidebar">
    **Collapsible Sidebar**

    - Click the menu toggle to collapse/expand
    - Collapsed shows icons only (hover for labels)
    - Preference saved per user
  </Tab>

  <Tab title="Widgets">
    **Widget Arrangement**

    - Drag and drop widgets to reorder (if enabled)
    - Hide/show specific widgets in settings
    - Reset to default layout anytime
  </Tab>

  <Tab title="Theme">
    **Visual Theme**

    - Light mode (default)
    - Dark mode (coming soon)
    - Follows system preference (coming soon)
  </Tab>
</Tabs>

### Personalization

In **Profile > Preferences**, customize:

- **Default Landing Page**: Dashboard, Solo Training, or Last Visited
- **Language**: German or English
- **Date Format**: DD/MM/YYYY or MM/DD/YYYY
- **Time Zone**: Auto-detected or manual selection
- **Notifications**: Which events trigger alerts

## Responsive Design

crisisIQ adapts to your screen size:

### Desktop (1280px+)

- Full sidebar navigation
- Multi-column layouts
- All widgets visible
- Data tables with all columns

### Tablet (768px - 1279px)

- Collapsible sidebar
- Two-column layouts
- Essential widgets shown
- Scrollable tables

### Mobile (< 768px)

- Hamburger menu
- Single-column layouts
- Priority widgets only
- Simplified tables
- Touch-optimized controls

## Dashboard Performance

### Loading States

While data loads, you'll see:

- **Skeleton Screens**: Placeholder shapes for content
- **Progress Bars**: For large data sets
- **Loading Spinners**: For quick operations

### Data Refresh

- **Auto-Refresh**: Dashboard updates every 30 seconds (configurable)
- **Manual Refresh**: Pull-to-refresh on mobile or refresh button
- **Real-Time Updates**: WebSocket connection for live simulations

### Offline Mode

When installed as PWA:

- **Cached Dashboard**: View previously loaded data
- **Offline Indicator**: Banner showing connection status
- **Sync on Reconnect**: Updates applied when back online

## Dashboard Navigation Tips

<AccordionGroup>
  <Accordion title="Keyboard Shortcuts">
    Speed up navigation with keyboard shortcuts:

    - `Ctrl/Cmd + K`: Open search
    - `Ctrl/Cmd + B`: Toggle sidebar
    - `Ctrl/Cmd + H`: Go to dashboard home
    - `Ctrl/Cmd + S`: Save current form (where applicable)
    - `Esc`: Close modals/dialogs
  </Accordion>

  <Accordion title="Breadcrumb Navigation">
    Use breadcrumbs at the top to navigate back:

    - Click any breadcrumb to jump to that level
    - Useful when deep in nested pages
    - Shows your current location
  </Accordion>

  <Accordion title="Company Switching">
    Quickly switch between organizations:

    1. Click Company Switcher dropdown
    2. Select different company
    3. Dashboard reloads with that company's data
    4. Your role may differ per company
  </Accordion>

  <Accordion title="Search Functionality">
    Find what you need fast:

    - Search scenarios by name, description, tags
    - Find users by name or email (Admin)
    - Locate simulations by title or ID
    - Filter results by type or date
  </Accordion>
</AccordionGroup>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Dashboard not loading">
    1. Check your internet connection
    2. Refresh the page (Ctrl/Cmd + R)
    3. Clear browser cache
    4. Try a different browser
    5. Contact support if issue persists
  </Accordion>

  <Accordion title="Widgets showing old data">
    - Click the refresh icon if available
    - Log out and back in
    - Check if real-time connection is active (look for connection indicator)
  </Accordion>

  <Accordion title="Can't see expected features">
    - Verify your role and permissions
    - Switch to the correct company context
    - Check if email is verified
    - Contact your administrator
  </Accordion>

  <Accordion title="Sidebar disappeared">
    - Click the menu toggle (hamburger icon) to show it
    - On mobile, sidebar is hidden by default - tap menu to open
  </Accordion>
</AccordionGroup>
