# Dashboard-Übersicht

Das crisisIQ-Dashboard ist Ihre zentrale Anlaufstelle für den Zugriff auf Simulationen, die Verwaltung von Inhalten und die Verfolgung Ihres Fortschritts.

## Dashboard-Typen

crisisIQ bietet verschiedene Dashboards basierend auf Ihrer Rolle und Aktivität:

<CardGroup cols={2}>
  <Card title="Haupt-Dashboard" icon="gauge">
    Primäre Zentrale für Administratoren, Content-Ersteller und Teammitglieder
  </Card>
  <Card title="Solo-Dashboard" icon="user">
    Dedizierte Oberfläche für individuelle Trainingsübungen
  </Card>
  <Card title="Teilnehmeransicht" icon="users">
    Vereinfachte Oberfläche beim Beitritt zu Gruppensimulationen
  </Card>
  <Card title="Admin-Panel" icon="shield">
    Erweiterte Funktionen für Unternehmens- und Systemadministratoren
  </Card>
</CardGroup>

## Haupt-Dashboard-Layout

Wenn Sie sich anmelden, sehen Sie das Haupt-Dashboard mit folgenden Hauptbereichen:

### Header-Navigation

<Tabs>
  <Tab title="Linke Seite">
    **Branding & Navigation**

    - **crisisIQ-Logo**: Klicken Sie, um zur Dashboard-Startseite zurückzukehren
    - **Menü-Toggle**: Seitenleiste ein-/ausblenden (mobil)
    - **Breadcrumbs**: Aktuelle Position innerhalb der App
  </Tab>

  <Tab title="Mitte">
    **Kontext & Suche**

    - **Unternehmenswechsler**: Aktive Organisation wechseln (wenn Sie zu mehreren gehören)
    - **Suchleiste**: Schnellsuche nach Szenarien, Benutzern oder Inhalten (nur Admin)
  </Tab>

  <Tab title="Rechte Seite">
    **Benutzersteuerung**

    - **Sprachauswahl**: Zwischen Deutsch (DE) und Englisch (EN) wechseln
    - **Benachrichtigungen**: Glockensymbol mit Anzahl ungelesener Nachrichten
    - **Profilmenü**: Zugriff auf Einstellungen, Abmelden, Hilfe
  </Tab>
</Tabs>

### Hauptinhaltsbereich

Der zentrale Bereich zeigt seitenspezifische Inhalte an:

- **Widgets**: Schnellstatistiken, bevorstehende Simulationen, aktuelle Aktivitäten
- **Datentabellen**: Simulationsliste, Teilnehmer, Szenarien
- **Formulare**: Inhalte erstellen/bearbeiten
- **Berichte**: Diagramme, Grafiken, Leistungskennzahlen

### Fußzeile (Optional)

Einige Seiten zeigen eine Fußzeile mit:
- Versionsinformationen
- Link zur Datenschutzrichtlinie
- Link zu Nutzungsbedingungen
- Support-Kontakt

## Dashboard-Anpassung

### Layout-Optionen

<Tabs>
  <Tab title="Seitenleiste">
    **Einklappbare Seitenleiste**

    - Klicken Sie auf den Menü-Toggle zum Ein-/Ausklappen
    - Eingeklappt zeigt nur Symbole (Hover für Beschriftungen)
    - Präferenz wird pro Benutzer gespeichert
  </Tab>

  <Tab title="Widgets">
    **Widget-Anordnung**

    - Widgets per Drag & Drop neu anordnen (falls aktiviert)
    - Spezifische Widgets in den Einstellungen ein-/ausblenden
    - Jederzeit auf Standard-Layout zurücksetzen
  </Tab>

  <Tab title="Theme">
    **Visuelles Theme**

    - Heller Modus (Standard)
    - Dunkler Modus (in Kürze verfügbar)
    - Folgt Systempräferenz (in Kürze verfügbar)
  </Tab>
</Tabs>

### Personalisierung

In **Profil > Einstellungen** anpassen:

- **Standard-Landingpage**: Dashboard, Solo-Training oder Zuletzt besucht
- **Sprache**: Deutsch oder Englisch
- **Datumsformat**: DD/MM/YYYY oder MM/DD/YYYY
- **Zeitzone**: Automatisch erkannt oder manuelle Auswahl
- **Benachrichtigungen**: Welche Ereignisse Alarme auslösen

## Responsive Design

crisisIQ passt sich Ihrer Bildschirmgröße an:

### Desktop (&gt; 1280px)

- Vollständige Seitenleisten-Navigation
- Mehrspaltige Layouts
- Alle Widgets sichtbar
- Datentabellen mit allen Spalten

### Tablet (768px - 1279px)

- Einklappbare Seitenleiste
- Zweispaltige Layouts
- Wesentliche Widgets angezeigt
- Scrollbare Tabellen

### Mobil (&lt; 768px)

- Hamburger-Menü
- Einspaltige Layouts
- Nur prioritäre Widgets
- Vereinfachte Tabellen
- Touch-optimierte Steuerelemente

<Tip>
Installieren Sie crisisIQ als Progressive Web App (PWA) auf Mobilgeräten für die beste Erfahrung. Siehe unseren [Mobile Installationshandbuch](/user-manual/mobile/ios-installation).
</Tip>

## Dashboard-Leistung

### Ladezustände

Während Daten geladen werden, sehen Sie:

- **Skeleton-Screens**: Platzhalterformen für Inhalte
- **Fortschrittsbalken**: Für große Datensätze
- **Lade-Spinner**: Für schnelle Operationen

### Datenaktualisierung

- **Auto-Refresh**: Dashboard wird alle 30 Sekunden aktualisiert (konfigurierbar)
- **Manuelle Aktualisierung**: Pull-to-Refresh auf Mobilgeräten oder Aktualisierungsschaltfläche
- **Echtzeit-Updates**: WebSocket-Verbindung für Live-Simulationen

### Offline-Modus

Wenn als PWA installiert:

- **Zwischengespeichertes Dashboard**: Zuvor geladene Daten anzeigen
- **Offline-Indikator**: Banner zeigt Verbindungsstatus
- **Synchronisierung bei Wiederverbindung**: Updates werden angewendet, wenn wieder online

## Dashboard-Navigationstipps

<AccordionGroup>
  <Accordion title="Tastaturkürzel">
    Beschleunigen Sie die Navigation mit Tastaturkürzeln:

    - `Strg/Cmd + K`: Suche öffnen
    - `Strg/Cmd + B`: Seitenleiste umschalten
    - `Strg/Cmd + H`: Zur Dashboard-Startseite gehen
    - `Strg/Cmd + S`: Aktuelles Formular speichern (wo zutreffend)
    - `Esc`: Modale/Dialoge schließen
  </Accordion>

  <Accordion title="Breadcrumb-Navigation">
    Verwenden Sie Breadcrumbs oben, um zurück zu navigieren:

    - Klicken Sie auf einen beliebigen Breadcrumb, um zu dieser Ebene zu springen
    - Nützlich, wenn Sie sich tief in verschachtelten Seiten befinden
    - Zeigt Ihre aktuelle Position
  </Accordion>

  <Accordion title="Unternehmenswechsel">
    Schnell zwischen Organisationen wechseln:

    1. Klicken Sie auf Unternehmenswechsler-Dropdown
    2. Wählen Sie ein anderes Unternehmen aus
    3. Dashboard lädt mit den Daten dieses Unternehmens neu
    4. Ihre Rolle kann pro Unternehmen unterschiedlich sein
  </Accordion>

  <Accordion title="Suchfunktionalität">
    Finden Sie schnell, was Sie brauchen:

    - Szenarien nach Name, Beschreibung, Tags suchen
    - Benutzer nach Name oder E-Mail finden (Admin)
    - Simulationen nach Titel oder ID lokalisieren
    - Ergebnisse nach Typ oder Datum filtern
  </Accordion>
</AccordionGroup>

## Fehlerbehebung

<AccordionGroup>
  <Accordion title="Dashboard lädt nicht">
    1. Überprüfen Sie Ihre Internetverbindung
    2. Aktualisieren Sie die Seite (Strg/Cmd + R)
    3. Browser-Cache löschen
    4. Versuchen Sie einen anderen Browser
    5. Kontaktieren Sie den Support, wenn das Problem weiterhin besteht
  </Accordion>

  <Accordion title="Widgets zeigen alte Daten">
    - Klicken Sie auf das Aktualisierungssymbol, falls verfügbar
    - Melden Sie sich ab und wieder an
    - Überprüfen Sie, ob die Echtzeit-Verbindung aktiv ist (suchen Sie nach Verbindungsindikator)
  </Accordion>

  <Accordion title="Erwartete Funktionen nicht sichtbar">
    - Überprüfen Sie Ihre Rolle und Berechtigungen
    - Wechseln Sie zum korrekten Unternehmenskontext
    - Überprüfen Sie, ob die E-Mail verifiziert ist
    - Kontaktieren Sie Ihren Administrator
  </Accordion>

  <Accordion title="Seitenleiste verschwunden">
    - Klicken Sie auf den Menü-Toggle (Hamburger-Symbol), um sie anzuzeigen
    - Auf Mobilgeräten ist die Seitenleiste standardmäßig ausgeblendet - tippen Sie auf Menü zum Öffnen
  </Accordion>
</AccordionGroup>
