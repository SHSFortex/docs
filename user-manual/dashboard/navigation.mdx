# Dashboard Navigation

Learn how to efficiently navigate through crisisIQ and access all features available to you.

## Navigation Menu

The main navigation menu is located in the left sidebar and provides access to all major features.

### Menu Structure

<Tabs>
  <Tab title="Core Features">
    **Available to All Users:**

    - **Dashboard**: Home page with overview and quick actions
    - **My Simulations**: Assigned and active training exercises
    - **Solo Training**: Individual practice scenarios
    - **Reports**: Your performance and progress
    - **Profile**: Account settings and preferences
  </Tab>

  <Tab title="Content Creation">
    **For Content Creators:**

    - **Scenarios**: Scenario library and builder
    - **Timeline Builder**: Create event sequences
    - **Resources**: Media library for scenarios
  </Tab>

  <Tab title="Administration">
    **For Administrators:**

    - **Company Management**: Organization settings
    - **User Management**: Invite and manage users
    - **Assignments**: Assign simulations to users/teams
    - **Analytics**: Organization-wide reports
  </Tab>

  <Tab title="System Admin">
    **For System Administrators:**

    - **All Companies**: Manage multiple organizations
    - **System Settings**: Platform configuration
    - **Audit Logs**: Security and activity logs
  </Tab>
</Tabs>

## Navigation Patterns

### Sidebar Navigation

<Steps>
  <Step title="Click Menu Item">
    Select the feature you want to access from the sidebar
  </Step>

  <Step title="View Submenu (if applicable)">
    Some items expand to show sub-options
  </Step>

  <Step title="Content Loads">
    Main area updates with the selected page
  </Step>

  <Step title="Active Indicator">
    Current page is highlighted in the sidebar
  </Step>
</Steps>

### Breadcrumbs

Navigate back through page hierarchy:

```
Dashboard > Scenarios > Timeline Builder > Edit Event
```

- Click any level to jump back
- Shows your current location
- Helps prevent getting lost in nested pages

### Back Button

Use your browser's back button or:

- In-app back button (when available)
- Breadcrumb navigation
- Sidebar menu to jump to different section

## Company Switcher

If you belong to multiple organizations:

<Steps>
  <Step title="Locate Company Switcher">
    Find the dropdown in the top navigation bar (shows current company name)
  </Step>

  <Step title="Click to Expand">
    View list of all organizations you belong to
  </Step>

  <Step title="Review Your Roles">
    Each company shows your role (Admin, Creator, Participant)
  </Step>

  <Step title="Select Company">
    Click the organization you want to work with
  </Step>

  <Step title="Dashboard Updates">
    Page reloads with the selected company's context
  </Step>
</Steps>

<Note>
Switching companies changes your available features based on your role in that organization.
</Note>

## Language Switcher

Change the interface language instantly:

1. Click the **language flag icon** in the top-right
2. Select **Deutsch (DE)** or **English (EN)**
3. Interface updates immediately
4. Preference is saved to your account

## Search Functionality

Quickly find content using the search bar:

### Global Search

<Steps>
  <Step title="Click Search Bar">
    Located in the top navigation (or press `Ctrl/Cmd + K`)
  </Step>

  <Step title="Type Query">
    Enter scenario name, user email, or simulation title
  </Step>

  <Step title="View Results">
    Results appear in real-time as you type
  </Step>

  <Step title="Filter Results">
    Use tabs to filter by type (Scenarios, Users, Simulations)
  </Step>

  <Step title="Select Result">
    Click any result to navigate to that page
  </Step>
</Steps>

### Search Tips

<Check>Use quotes for exact phrases: `"crisis management"`</Check>
<Check>Filter by type: `type:scenario` or `type:user`</Check>
<Check>Search by ID: `SIM-12345`</Check>
<Check>Use wildcards: `train*` matches "training", "trainer", etc.</Check>

## Mobile Navigation

On mobile devices, navigation adapts for touch:

### Hamburger Menu

- Tap the **☰** icon to open sidebar
- Sidebar slides in from left
- Tap outside or ✕ to close

### Bottom Navigation (Optional)

Some views show quick actions at the bottom:

- Home
- Simulations
- Profile
- More

### Swipe Gestures

- **Swipe right**: Open sidebar menu
- **Swipe left**: Close sidebar menu
- **Pull down**: Refresh current page

## Keyboard Shortcuts

Power users can navigate faster with keyboard shortcuts:

<Tabs>
  <Tab title="General">
    **Navigation Shortcuts:**

    - `Ctrl/Cmd + K`: Open search
    - `Ctrl/Cmd + H`: Go to dashboard
    - `Ctrl/Cmd + B`: Toggle sidebar
    - `Esc`: Close modal/drawer
  </Tab>

  <Tab title="Editing">
    **When Creating/Editing:**

    - `Ctrl/Cmd + S`: Save
    - `Ctrl/Cmd + Enter`: Save and close
    - `Esc`: Cancel/close without saving
  </Tab>

  <Tab title="Tables">
    **In Data Tables:**

    - `Arrow Keys`: Navigate cells
    - `Enter`: Edit selected row
    - `Delete`: Delete selected row (with confirmation)
    - `Ctrl/Cmd + A`: Select all
  </Tab>
</Tabs>

## Contextual Actions

Actions appear based on your current context:

### Quick Actions Bar

Located at the top of list views:

- **+ Create**: Start new item (scenario, simulation, etc.)
- **Filter**: Narrow results
- **Sort**: Change ordering
- **View Options**: Grid, list, table views

### Row Actions

On table rows, hover to reveal:

- **View**: See details
- **Edit**: Modify item
- **Duplicate**: Create copy
- **Delete**: Remove (with confirmation)

### Floating Action Button (FAB)

On mobile, a circular **+** button provides quick access to create actions.

## Notifications Center

Access via the bell icon in the top-right:

<Steps>
  <Step title="Click Bell Icon">
    Badge shows unread count
  </Step>

  <Step title="View Notifications">
    Panel slides out with recent notifications
  </Step>

  <Step title="Filter by Type">
    Tabs for All, Simulations, Administrative, Social
  </Step>

  <Step title="Take Action">
    Click notification to view details or mark as read
  </Step>

  <Step title="Manage">
    Mark all as read or configure notification settings
  </Step>
</Steps>

## Page-Specific Navigation

### Scenario Editor

Specialized navigation for scenario building:

- **Tabs**: Overview, Timeline, Decisions, Resources
- **Preview Mode**: See scenario as participants will
- **Save Progress**: Autosave every 30 seconds
- **Exit**: Return to scenario library

### Simulation Control

Navigate simulation phases:

- **Preparation**: Setup and configure
- **Briefing**: Participant instructions
- **Execution**: Live simulation running
- **Debriefing**: Review and feedback
- **Reports**: Performance analysis

### Timeline Builder

Visual navigation for event sequencing:

- **Zoom Controls**: In/out on timeline
- **Pan**: Drag timeline left/right
- **Event Cards**: Click to edit
- **Add Event**: Click on timeline to insert

## Accessibility Navigation

crisisIQ supports assistive navigation:

### Screen Readers

- **Skip to Content**: Bypass navigation
- **Landmark Regions**: Proper ARIA landmarks
- **Focus Management**: Logical tab order
- **Announcements**: Important updates spoken

### Keyboard-Only Navigation

- **Tab**: Move between interactive elements
- **Shift + Tab**: Move backwards
- **Enter/Space**: Activate buttons
- **Arrow Keys**: Navigate menus and lists

### High Contrast Mode

Respects system high contrast settings for better visibility.
