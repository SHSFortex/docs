# E-Mail-Verifizierung

Die E-Mail-Verifizierung gewährleistet die Sicherheit Ihres Kontos und ermöglicht vollen Zugriff auf crisisIQ-Funktionen.

## Warum E-Mail verifizieren?

Die E-Mail-Verifizierung ist erforderlich, um:

<CardGroup cols={2}>
  <Card title="Sicherheit" icon="shield-check">
    Bestätigt, dass Sie der Besitzer der E-Mail-Adresse sind
  </Card>
  <Card title="Kommunikation" icon="envelope">
    Ermöglicht wichtige Benachrichtigungen und Updates
  </Card>
  <Card title="Passwortwiederherstellung" icon="key">
    Ermöglicht das Zurücksetzen des Passworts, falls Sie es vergessen
  </Card>
  <Card title="Vollzugriff" icon="unlock">
    Erforderlich für die Teilnahme an Simulationen
  </Card>
</CardGroup>

<Warning>
Einige Funktionen, insbesondere **die Teilnahme an Teilnehmersimulationen**, erfordern eine verifizierte E-Mail-Adresse. Möglicherweise sehen Sie eingeschränkte Funktionen, bis die Verifizierung abgeschlossen ist.
</Warning>

## Verifizierungsvorgang

### Für neue Konten

Wenn Sie ein neues Konto erstellen:

<Steps>
  <Step title="Konto registrieren">
    Füllen Sie das Registrierungsformular mit Ihrer E-Mail und anderen Details aus
  </Step>

  <Step title="E-Mail überprüfen">
    Suchen Sie in Ihrem Posteingang nach einer Verifizierungs-E-Mail von crisisIQ
  </Step>

  <Step title="Verifizierungslink klicken">
    Öffnen Sie die E-Mail und klicken Sie auf den Button oder Link **"E-Mail-Adresse verifizieren"**
  </Step>

  <Step title="Bestätigung">
    Sie werden mit einer Erfolgsmeldung zu crisisIQ weitergeleitet
  </Step>

  <Step title="Zugriff gewährt">
    Ihr Konto ist jetzt vollständig aktiviert und Sie können auf alle Funktionen zugreifen
  </Step>
</Steps>

### Für eingeladene Benutzer

Wenn Sie von einer Organisation eingeladen wurden:

1. Akzeptieren Sie die Einladung über den E-Mail-Link
2. Legen Sie Ihr Passwort fest
3. Eine Verifizierungs-E-Mail wird automatisch gesendet
4. Klicken Sie auf den Verifizierungslink, um die Einrichtung abzuschließen

## Verifizierungs-E-Mail

### Was Sie erwarten können

Die Verifizierungs-E-Mail enthält:

- **Betreff**: "Verifizieren Sie Ihre crisisIQ-E-Mail-Adresse"
- **Von**: noreply@crisisiq.com
- **Verifizierungslink**: Eindeutige, zeitlich begrenzte URL
- **Gültigkeit**: Link läuft nach **24 Stunden** ab

### E-Mail-Beispiel

```
Betreff: Verifizieren Sie Ihre crisisIQ-E-Mail-Adresse

Hallo [Ihr Name],

Willkommen bei crisisIQ! Bitte verifizieren Sie Ihre E-Mail-Adresse, um Ihr Konto zu aktivieren.

[Button "E-Mail-Adresse verifizieren"]

Oder kopieren Sie diesen Link und fügen Sie ihn ein:
https://crisisiq.com/verify-email?token=abc123...

Dieser Link läuft in 24 Stunden ab.

Wenn Sie dieses Konto nicht erstellt haben, ignorieren Sie bitte diese E-Mail.

- Das crisisIQ-Team
```

## Verifizierungs-E-Mail erneut senden

Wenn Sie die Verifizierungs-E-Mail nicht erhalten haben oder sie abgelaufen ist:

<Steps>
  <Step title="Anmeldeversuch">
    Versuchen Sie, sich bei Ihrem crisisIQ-Konto anzumelden
  </Step>

  <Step title="Verifizierungshinweis">
    Sie sehen eine Meldung: **"E-Mail-Verifizierung erforderlich"**
  </Step>

  <Step title="Erneut senden anfordern">
    Klicken Sie auf den Button **"Verifizierungs-E-Mail erneut senden"**
  </Step>

  <Step title="Posteingang überprüfen">
    Eine neue Verifizierungs-E-Mail wird innerhalb von Minuten gesendet
  </Step>

  <Step title="Verifizieren">
    Klicken Sie auf den Link in der neuen E-Mail, um die Verifizierung abzuschließen
  </Step>
</Steps>

<Note>
Sie können bis zu **5 Mal pro Stunde** eine neue Verifizierungs-E-Mail anfordern, um Missbrauch zu verhindern.
</Note>

## E-Mail kommt nicht an?

Wenn die Verifizierungs-E-Mail nicht in Ihrem Posteingang erscheint:

<AccordionGroup>
  <Accordion title="Spam-/Junk-Ordner überprüfen">
    E-Mail-Filter haben die Nachricht möglicherweise umgeleitet. Überprüfen Sie:
    - Spam-Ordner
    - Junk-Ordner
    - Werbung-Tab (Gmail)
    - Updates-Tab (Gmail)
  </Accordion>

  <Accordion title="Ein paar Minuten warten">
    Die E-Mail-Zustellung kann 2-5 Minuten dauern. Haben Sie Geduld, bevor Sie ein erneutes Senden anfordern.
  </Accordion>

  <Accordion title="E-Mail-Adresse überprüfen">
    Melden Sie sich ab und versuchen Sie, sich erneut anzumelden. Das System zeigt Ihnen, welche E-Mail-Adresse verifiziert werden muss. Stellen Sie sicher, dass sie korrekt ist.
  </Accordion>

  <Accordion title="Absender auf Whitelist setzen">
    Fügen Sie diese Adressen zu Ihrer E-Mail-Whitelist/sicheren Absendern hinzu:
    - noreply@crisisiq.com
    - support@crisisiq.com
  </Accordion>

  <Accordion title="E-Mail-Filter überprüfen">
    Einige Unternehmens-E-Mail-Systeme haben aggressive Filter. Kontaktieren Sie Ihre IT-Abteilung, um crisisIQ-E-Mails auf die Whitelist zu setzen.
  </Accordion>

  <Accordion title="Andere E-Mail-Adresse versuchen">
    Wenn Sie einen Wegwerf- oder temporären E-Mail-Dienst verwendet haben, empfängt dieser möglicherweise keine Nachrichten. Kontaktieren Sie den Support, um Ihre Konto-E-Mail zu aktualisieren.
  </Accordion>
</AccordionGroup>

## Verifizierungsstatus

### Status überprüfen

Um zu sehen, ob Ihre E-Mail verifiziert ist:

1. Melden Sie sich bei crisisIQ an (wenn möglich)
2. Gehen Sie zu **Profileinstellungen**
3. Suchen Sie nach dem Indikator **E-Mail-Status**

<Tabs>
  <Tab title="Verifiziert ✓">
    **Status**: E-Mail verifiziert

    - Grünes Häkchen neben der E-Mail
    - Vollzugriff auf alle Funktionen
    - Kann an Simulationen teilnehmen
    - Erhält alle Benachrichtigungen
  </Tab>

  <Tab title="Ausstehend ✗">
    **Status**: Verifizierung ausstehend

    - Orangefarbenes Warnsymbol
    - Eingeschränkter Funktionszugriff
    - Kann nicht an Teilnehmersimulationen teilnehmen
    - Option zum erneuten Senden der Verifizierung verfügbar
  </Tab>
</Tabs>

### Einschränkungen bei nicht verifiziertem Konto

Bis Sie Ihre E-Mail verifizieren, können Sie Folgendes erleben:

<Warning>
- **Kann nicht an Simulationen teilnehmen** als Teilnehmer
- **Eingeschränkte Benachrichtigungen** (nur in der App)
- **Keine Passwortzurücksetzung** möglich
- **Eingeschränkte Einladungen** an andere Benutzer
</Warning>

Administrations- und Content-Erstellungsfunktionen können je nach den Einstellungen Ihrer Organisation weiterhin zugänglich sein.

## E-Mail-Adresse ändern

Wenn Sie Ihre E-Mail-Adresse ändern müssen:

<Steps>
  <Step title="Im Profil aktualisieren">
    Gehen Sie zu Profileinstellungen und aktualisieren Sie Ihre E-Mail-Adresse
  </Step>

  <Step title="Neue Verifizierung erforderlich">
    Eine Verifizierungs-E-Mail wird an die neue Adresse gesendet
  </Step>

  <Step title="Neue E-Mail verifizieren">
    Klicken Sie auf den Link in der E-Mail, die an Ihre neue Adresse gesendet wurde
  </Step>

  <Step title="Alte E-Mail deaktiviert">
    Ihre alte E-Mail wird ersetzt, sobald die neue verifiziert ist
  </Step>
</Steps>

<Info>
Ihre alte E-Mail bleibt aktiv, bis Sie die neue verifiziert haben. Dies verhindert versehentliche Aussperrung.
</Info>

## Fehlerbehebung

<AccordionGroup>
  <Accordion title="Verifizierungslink sagt 'abgelaufen'">
    Links laufen nach 24 Stunden ab. Melden Sie sich an und fordern Sie eine neue Verifizierungs-E-Mail von der Verifizierungs-Aufforderungsseite an.
  </Accordion>

  <Accordion title="Verifizierungslink sagt 'ungültig'">
    Der Link wurde möglicherweise bereits verwendet oder ist fehlerhaft. Fordern Sie eine neue Verifizierungs-E-Mail an.
  </Accordion>

  <Accordion title="Klicken auf den Link bewirkt nichts">
    - Versuchen Sie, die vollständige URL in Ihren Browser zu kopieren und einzufügen
    - Stellen Sie sicher, dass Sie einen aktuellen Browser verwenden
    - Überprüfen Sie, ob Browser-Erweiterungen stören
    - Versuchen Sie es in einem anderen Browser oder Inkognito-Modus
  </Accordion>

  <Accordion title="Zeigt nach Klicken immer noch unverifiziert an">
    - Löschen Sie Ihren Browser-Cache und Cookies
    - Melden Sie sich ab und wieder an
    - Warten Sie einige Minuten, bis das System aktualisiert wird
    - Kontaktieren Sie den Support, wenn das Problem weiterhin besteht
  </Accordion>

  <Accordion title="Kann nicht auf E-Mail-Konto zugreifen">
    Wenn Sie keinen Zugriff mehr auf Ihre registrierte E-Mail haben:
    - Kontaktieren Sie Ihren Organisationsadministrator
    - Er kann Ihre E-Mail-Adresse aktualisieren
    - Sie erhalten eine Verifizierungs-E-Mail an die neue Adresse
  </Accordion>

  <Accordion title="Unternehmens-E-Mail blockiert Nachrichten">
    Einige Unternehmens-E-Mail-Systeme blockieren automatisierte E-Mails:
    - Bitten Sie Ihre IT-Abteilung, crisisIQ auf die Whitelist zu setzen
    - Geben Sie ihnen: noreply@crisisiq.com und support@crisisiq.com
    - Alternativ können Sie eine persönliche E-Mail verwenden, falls von Ihrer Organisation erlaubt
  </Accordion>
</AccordionGroup>
