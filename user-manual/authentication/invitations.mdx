# Einladungen & Beitrittscodes

Erfahren Sie, wie Sie crisisIQ über E-Mail-Einladungen oder Organisations-Beitrittscodes beitreten können.

## Arten von Einladungen

Es gibt drei Möglichkeiten, einer crisisIQ-Organisation beizutreten:

  <Card title="E-Mail-Einladung" icon="envelope">

  <Card title="Beitrittscode" icon="key">

## E-Mail-Einladungen

Organisationsadministratoren können Sie per E-Mail einladen, ihrer crisisIQ-Organisation beizutreten.

### Einladung erhalten

Wenn Sie eingeladen werden, erhalten Sie eine E-Mail mit:

- **Betreff**: "Sie wurden eingeladen, [Firmenname] bei crisisIQ beizutreten"
- **Von**: noreply@crisisiq.com
- **Einladungsdetails**: Firmenname, Ihre zugewiesene Rolle, Name des Einladenden
- **Akzeptierungslink**: Eindeutige URL zur Annahme der Einladung
- **Ablauf**: Einladung gültig für **7 Tage**

### E-Mail-Einladung annehmen

<Steps>
  <Step title="E-Mail überprüfen">
    Suchen Sie nach der Einladungs-E-Mail (überprüfen Sie auch Spam-/Junk-Ordner)
  </Step>

  <Step title="Details überprüfen">
    Lesen Sie die Einladung, um zu verstehen:
    - Welche Organisation Sie einlädt
    - Ihre zugewiesene Rolle (Teilnehmer, Content-Ersteller, Administrator)
    - Wer die Einladung gesendet hat
  </Step>

  <Step title="Auf 'Einladung annehmen' klicken">
    Klicken Sie auf den Akzeptierungslink in der E-Mail
  </Step>

  <Step title="Konto erstellen/Anmelden">
    **Wenn Sie kein Konto haben:**
    - Geben Sie Ihren Namen ein
    - Erstellen Sie ein Passwort
    - Akzeptieren Sie die Nutzungsbedingungen

    **Wenn Sie bereits ein Konto haben:**
    - Melden Sie sich mit Ihren bestehenden Anmeldedaten an
    - Die Organisation wird Ihrem Konto hinzugefügt
  </Step>

  <Step title="E-Mail verifizieren (bei neuem Konto)">
    Schließen Sie die E-Mail-Verifizierung ab, falls erforderlich
  </Step>

  <Step title="Auf Organisation zugreifen">
    Sie werden zu Ihrem Dashboard mit Zugriff auf die neue Organisation weitergeleitet
  </Step>
</Steps>

<Info>
Wenn Sie bereits ein crisisIQ-Konto haben, fügt das Annehmen einer Einladung die neue Organisation zu Ihrem bestehenden Konto hinzu. Sie können mit dem Firmenwechsler zwischen Organisationen wechseln.
</Info>

## Beitrittscodes

Beitrittscodes bieten eine schnelle Möglichkeit, einer Organisation beizutreten, ohne eine direkte E-Mail-Einladung zu benötigen.

### Was ist ein Beitrittscode?

- **Format**: 8-12 Zeichen alphanumerischer Code (z.B. `ABC123XY`)
- **Geteilt von**: Organisationsadministratoren
- **Zweck**: Massen-Onboarding, Workshop-Teilnehmer, temporärer Zugang
- **Ablauf**: Kann vom Administrator festgelegt werden (Standard: kein Ablauf)

### Beitrittscode verwenden

<Steps>
  <Step title="Zur Beitrittsseite navigieren">
    Klicken Sie auf dem crisisIQ-Anmeldebildschirm auf **"Mit Code beitreten"**

    Oder besuchen Sie: `https://crisisiq.com/join-with-code`
  </Step>

  <Step title="Code eingeben">
    Geben Sie den von Ihrer Organisation bereitgestellten Beitrittscode ein oder fügen Sie ihn ein
  </Step>

  <Step title="Auf 'Code überprüfen' klicken">
    Das System validiert den Code und zeigt den Organisationsnamen an
  </Step>

  <Step title="Ihre Daten angeben">
    **Wenn Sie kein Konto haben:**
    - Geben Sie Ihren vollständigen Namen ein
    - Geben Sie Ihre E-Mail-Adresse an
    - Erstellen Sie ein Passwort
    - Akzeptieren Sie die Nutzungsbedingungen

    **Wenn Sie ein Konto haben:**
    - Melden Sie sich mit Ihren bestehenden Anmeldedaten an
  </Step>

  <Step title="Beitritt bestätigen">
    Überprüfen Sie die Organisationsdetails und klicken Sie auf **"Organisation beitreten"**
  </Step>

  <Step title="E-Mail verifizieren (bei neuem Konto)">
    Überprüfen Sie Ihre E-Mail auf einen Verifizierungslink
  </Step>

  <Step title="crisisIQ verwenden">
    Greifen Sie auf das Dashboard und die Funktionen Ihrer neuen Organisation zu
  </Step>
</Steps>

### Vorteile von Beitrittscodes

<Check>**Schnelles Onboarding** - Keine Wartezeit auf E-Mail</Check>
<Check>**Massen-Registrierung** - Einen Code mit vielen Benutzern teilen</Check>
<Check>**Flexibles Teilen** - Per E-Mail, Chat, Präsentation usw.</Check>
<Check>**Keine E-Mail erforderlich** - Benutzer können entdecken und beitreten</Check>

## Simulationsspezifische Einladungen

Teilnehmer können direkte Links erhalten, um an einer bestimmten Simulation teilzunehmen.

### Simulationseinladungs-E-Mail

Wenn Sie zu einer Simulation eingeladen werden, erhalten Sie:

- **Betreff**: "Sie sind eingeladen, an [Simulationsname] teilzunehmen"
- **Von**: noreply@crisisiq.com
- **Simulationsdetails**: Titel, Beschreibung, Startzeit
- **Beitrittslink**: Direkter Link zur Simulation
- **Anforderungen**: Möglicherweise E-Mail-Verifizierung erforderlich zum Beitritt

### An einer Simulation teilnehmen

<Steps>
  <Step title="Simulationslink klicken">
    Öffnen Sie die Einladungs-E-Mail und klicken Sie auf den Link **"An Simulation teilnehmen"**
  </Step>

  <Step title="Anmelden oder Registrieren">
    - **Bestehende Benutzer**: Anmelden
    - **Neue Benutzer**: Konto erstellen
  </Step>

  <Step title="E-Mail verifizieren (falls erforderlich)">
    Einige Simulationen erfordern verifizierte E-Mail-Adressen
  </Step>

  <Step title="Auf Simulation zugreifen">
    Sie werden zur Vorbereitungs- oder Briefing-Seite der Simulation weitergeleitet
  </Step>

  <Step title="Auf Start warten">
    Folgen Sie dem Simulationszeitplan (Vorbereitung → Briefing → Durchführung → Nachbesprechung)
  </Step>
</Steps>

<Note>
Simulationseinladungen können bestimmte Startzeiten haben. Treten Sie früh bei, um die Vorbereitungsphase abzuschließen.
</Note>

## Einladungsstatus

### Ausstehende Einladungen überprüfen

Um Einladungen zu sehen, die Sie noch nicht angenommen haben:

1. Melden Sie sich bei Ihrem crisisIQ-Konto an
2. Suchen Sie nach einem Benachrichtigungsabzeichen oder Banner
3. Oder gehen Sie zu **Profil** > **Einladungen**
4. Überprüfen Sie ausstehende Einladungen und akzeptieren/ablehnen Sie sie

### Mehrere Organisationen verwalten

Wenn Sie mehreren Organisationen angehören:

<Tabs>
  <Tab title="Firmenwechsler">
    **Zwischen Organisationen wechseln**

    1. Klicken Sie auf den **Firmenwechsler** in der oberen Navigation
    2. Sehen Sie alle Organisationen, denen Sie angehören
    3. Wählen Sie die aus, mit der Sie arbeiten möchten
    4. Dashboard wird aktualisiert, um den Inhalt dieses Unternehmens anzuzeigen
  </Tab>

  <Tab title="Verschiedene Rollen">
    **Rollenbasierter Zugriff**

    - Sie können in verschiedenen Organisationen unterschiedliche Rollen haben
    - Berechtigungen und Funktionen variieren je nach Rolle
    - Firmenwechsler zeigt Ihre Rolle in jeder Organisation
  </Tab>
</Tabs>

## Fehlerbehebung bei Einladungen

<AccordionGroup>
  <Accordion title="Einladungs-E-Mail nicht erhalten">
    1. Spam-/Junk-Ordner überprüfen
    2. Überprüfen, ob der Administrator sie an die richtige E-Mail gesendet hat
    3. noreply@crisisiq.com auf Whitelist setzen
    4. Administrator bitten, die Einladung erneut zu senden
  </Accordion>

  <Accordion title="Einladungslink sagt 'abgelaufen'">
    E-Mail-Einladungen laufen nach 7 Tagen ab. Kontaktieren Sie den Administrator, um eine neue Einladung zu senden, oder fordern Sie stattdessen einen Beitrittscode an.
  </Accordion>

  <Accordion title="Einladungslink sagt 'ungültig'">
    - Die Einladung wurde möglicherweise widerrufen
    - Sie wurde möglicherweise bereits verwendet
    - Der Link könnte fehlerhaft sein (stellen Sie sicher, dass Sie die gesamte URL kopiert haben)
    - Fordern Sie eine neue Einladung von Ihrem Administrator an
  </Accordion>

  <Accordion title="Beitrittscode funktioniert nicht">
    - Überprüfen Sie, ob Sie den Code korrekt eingegeben haben (Groß-/Kleinschreibung beachten)
    - Überprüfen Sie, ob der Code abgelaufen ist
    - Fragen Sie den Administrator, ob der Code noch aktiv ist
    - Stellen Sie sicher, dass Sie sich in der richtigen crisisIQ-Instanz befinden
  </Accordion>

  <Accordion title="Habe bereits ein Konto mit anderer E-Mail">
    Wenn Sie eine Einladung mit einer anderen E-Mail annehmen möchten:
    - Erstellen Sie ein neues Konto mit der eingeladenen E-Mail, ODER
    - Kontaktieren Sie den Administrator, um die Einladung an Ihre bestehende E-Mail erneut zu senden
  </Accordion>

  <Accordion title="Einladung angenommen, aber Organisation nicht sichtbar">
    - Melden Sie sich ab und wieder an
    - Überprüfen Sie das Dropdown-Menü des Firmenwechslers
    - Kontaktieren Sie Ihren Administrator, um Ihren Zugriff zu überprüfen
    - Browser-Cache löschen und erneut versuchen
  </Accordion>
</AccordionGroup>

## Einladungen ablehnen

Wenn Sie eine Einladung erhalten, die Sie nicht annehmen möchten:

<Steps>
  <Step title="Einladungslink klicken">
    Öffnen Sie die E-Mail und klicken Sie auf den Link
  </Step>

  <Step title="'Ablehnen' wählen">
    Anstatt anzunehmen, klicken Sie auf **"Einladung ablehnen"**
  </Step>

  <Step title="Ablehnung bestätigen">
    Bestätigen Sie, dass Sie dieser Organisation nicht beitreten möchten
  </Step>

  <Step title="Benachrichtigung gesendet">
    Der Administrator wird möglicherweise über Ihre Ablehnung benachrichtigt (je nach Einstellungen)
  </Step>
</Steps>

<Note>
Das Ablehnen einer Einladung hindert Sie nicht daran, später über einen Beitrittscode oder eine neue Einladung beizutreten.
</Note>
