# Certificates & Achievements

Recognize and document your crisis management training accomplishments with certificates and achievement badges.

## Training Certificates

### What Are Certificates?

Official documents confirming:

### Earning Certificates

Criteria (scenario-dependent):

<Check>**Minimum Score**: Usually 70-80%+</Check>
<Check>**Full Completion**: All decisions made</Check>
<Check>**Required Elements**: Mandatory decisions answered</Check>
<Check>**Time Limits**: Completed within allowed time (if applicable)</Check>

<Note>
Not all scenarios offer certificates. Check scenario details to see if certificate is available.
</Note>

## Certificate Contents

### Standard Information

- **Participant Name**: Your full name
- **Scenario Title**: Which simulation completed
- **Completion Date**: When achieved
- **Score Achieved**: Your performance level
- **Skills Practiced**: Competencies demonstrated
- **Issuing Organization**: Company or crisisIQ
- **Verification Code**: Unique identifier for authentication

### Certificate Design

Professional format:

- Organization logo
- Official styling
- Digital signature (if configured)
- QR code for verification (optional)
- Suitable for printing or digital sharing

## Downloading Certificates

<Steps>
  <Step title="Complete Qualifying Simulation">

### File Format

- **PDF**: High-quality, printable
- **File Name**: YourName_ScenarioName_Date.pdf
- **Size**: Optimized for email/upload

Some organizations offer verification:

- **Verification URL**: Link to confirm authenticity
- **QR Code**: Scan to verify online

## Continuing Education Credits

### Professional Credits

Some professions accept crisisIQ training:

- **CPE** (Continuing Professional Education)
- **PDH** (Professional Development Hours)
- **CME** (Continuing Medical Education) - rare but possible
- **Industry-Specific** certifications

<Warning>
Check with your certification body whether crisisIQ training qualifies for credits. Requirements vary by profession and jurisdiction.
</Warning>

### Documenting Hours

Certificates typically show:

- **Contact Hours**: Time spent in simulation
- **Learning Objectives**: Skills addressed
- **Assessment**: Performance measurement
- **Provider Info**: crisisIQ or organization details

## Managing Certificates

### Organization

Keep certificates organized:

**Digital:**
- Dedicated folder (e.g., "Training Certificates")
- Cloud storage for backup
- Consistent file naming
- Indexed by date or skill

**Physical:**
- Binder or portfolio
- Chronological or by category
- Protected in sheet protectors
- Easily accessible for opportunities

### Updating Records

Maintain current records:

- Add new certificates as earned
- Update resume/LinkedIn
- Inform HR/management
- Track expiration dates (if applicable)

## Certificate Troubleshooting

<AccordionGroup>
  <Accordion title="No certificate available after passing">
    - Check scenario offers certificates (not all do)
    - Verify you met all criteria
    - Allow a few minutes for generation
    - Contact support if delayed
  </Accordion>

  <Accordion title="Can't download PDF">
    - Check browser pop-up blocker
    - Try different browser
    - Ensure Adobe Reader or PDF viewer installed
    - Right-click and "Save As"
  </Accordion>

  <Accordion title="Name misspelled on certificate">
    - Update your profile with correct name
    - Request certificate regeneration
    - Contact support for manual correction
  </Accordion>

  <Accordion title="Lost certificate file">
    - Return to simulation report
    - Download again (certificates persist)
    - If unavailable, contact support
  </Accordion>

  <Accordion title="Employer questions authenticity">
    - Provide verification code/URL
    - Direct to crisisIQ support for confirmation
    - Share simulation report as backup
  </Accordion>
</AccordionGroup>
